# Pomodoro Timer
>Pomodoro timer built with React.JS

## [Demo Version here](https://fenatan.github.io/timer-pomodoro)

### Features

- CountDown Timer
- User can change the minutes in session and break time before starting
- Audio Notification
- Toast Notification

### Running Locally
```sh
$ yarn add
$ yarn start
```
Runs the app in development mode.
Open http://localhost:3000 to view it in the browser.
